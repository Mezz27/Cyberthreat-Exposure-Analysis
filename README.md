# Cyberthreat Exposure Analysis

## Project Overview
This project analyzes global cyberthreat exposure using the VERIS Community Database (VCDB).

The goal is to identify:
- Most targeted industries
- Common attack vectors
- Geographic distribution of cyber incidents
- Trends over time

## Dataset
Source: Verizon Data Breach Investigations Report (DBIR)

Dataset used:
VERIS Community Database (VCDB)

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab

## Workflow
Clone → Parse → Analyze → Visualize → Export

## Analysis Performed

1. Industry Trends
Identifies industries most affected by cyber breaches.

2. Attack Patterns
Analyzes common attack methods such as:
- Phishing
- Malware
- Credential Theft

3. Geographic Insights
Shows breach distribution by country.

4. Time Trends
Tracks cyber incidents across years.

## Visualizations

### Top Industries Affected
Bar chart showing most targeted industries.

### Attack Vectors
Pie chart of common attack methods.

### Incidents Over Time
Line graph showing yearly trends.

## Output
The project exports:

- Cleaned CSV dataset
- Industry breach chart
- Attack vector distribution
- Yearly incident trend graph

## Example Insights
- Healthcare and finance are among the most targeted industries.
- Phishing is a leading attack vector.
- Cyber incidents show increasing trends over recent years.
